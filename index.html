Visual Studios is a app where we can code using html, js and css. First you have to download Visual Studios, then open it. Go to the last icon on the left hand side, extensions. Search Live Sever and download it. Now go to the first icon on the left hand side, Explorer. Create a folder on your desktop and come back to Visual Studios, click on 'File', open folder and then select the folder. Now create 3 files by right clicking on where you see you folders name. Name the files 'index.html', 'main.js' and 'style.css'. Now you can start coding.

We can on a console screen on any website by right clicking on the page and then click on inspect. Then a tab will open on the right side of the website. On the top of that tab click on console. To print something on a console screen you have to use the script tag.
